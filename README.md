# iris-net-pytorch

a simple neural network with python (pytorch)

---

how to use it?

1. clone the repository
```
git clone https://github.com/mobin-abdi/iris-net-pytorch.git
```

2. change directory
```
cd iris-net-pytorch
```

3. install dependencies
```
pip install -r requirements.txt
```

### NOTE: you must run it in jupyter-notebook
